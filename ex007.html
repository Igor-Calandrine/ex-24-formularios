Select

Usa-se select para abrir uma aba em que o usuário terá determinadas opções

Dentro do select coloca-se as opções com option.

Tamém pose-se dividir as opções em grupos com optgroup e distinguí-las com label dentro da tag.

Datalist

Pode-se oferecer ao usuário uma lista de possibilidades e sugestões já utilizadas pelo usuário.

Ocupação

TextArea

Uma área pode ser inserida para o usuário enviar um texto.

Recomenda-se a utilização do método POST devido a limitação de 3000 caracteres.
