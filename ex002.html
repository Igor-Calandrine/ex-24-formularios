GET

Nunca utilizar para envio de dados sensíveis

Os dados ficam na URL

Limite de 3000 bytes ou seja 3000 letras

Não envia imagens

POST

Não expõe os dados na URL

Usado para enviar dados confidenciais ou grandes

Não é cacheado por segurança

É possivel ainda encontrar os dados nas ferramentas do desenvolvedor, para que os dados não sejam visíveis eles devem ser criptografados.
